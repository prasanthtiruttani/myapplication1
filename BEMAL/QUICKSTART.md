## 📋 Quick Tasks Guide

### 🆕 Register a New Sample
1. Click **"Samples"** in navigation
2. Click **"+ Register Sample"** button
3. Fill in sample details:
   - Name: "Lead Acid Battery Plate A1"
   - Type: Select "Battery Plate"
   - Batch: "BAT-2024-001"
   - Quantity: "5 pieces"
   - Location: "Storage Room A1"
4. Click **"Register Sample"**
5. ✅ Sample appears in the table!

### 🔬 Schedule a Test
1. Click **"Tests"** in navigation
2. Click **"+ Schedule Test"** button
3. Fill in test details:
   - Sample ID: "SMP-001"
   - Test Type: Select "Capacity Test"
   - Date: Choose future date
   - Technician: "Lab Technician"
   - Equipment: "Battery Analyzer"
4. Click **"Schedule Test"**
5. ✅ Test appears in the schedule!

### ⚙️ Add Equipment
1. Click **"Equipment"** in navigation
2. Click **"+ Add Equipment"** button
3. Fill in equipment details:
   - Name: "Battery Analyzer BA-100"
   - Type: Select "Electrical Tester"
   - Location: "Test Lab A"
   - Status: "Available"
4. Click **"Add Equipment"**
5. ✅ Equipment appears in inventory!

### 📊 Generate a Report
1. Click **"Reports"** in navigation
2. Fill in report details:
   - Sample ID: "SMP-001"
   - Test ID: "TST-001"
   - Report Type: Select type
3. Click **"+ Generate Report"** button
4. ✅ Report appears in the table!

## 🧪 Sample Management

### Sample Types Available
- **Battery Plate**: Positive/negative plates
- **Electrolyte**: Sulfuric acid solutions
- **Separator**: AGM or PE separators
- **Terminal**: Battery terminals
- **Container**: Battery containers
- **Raw Material**: Lead, plastic, etc.
- **Finished Product**: Complete batteries

### Sample Status
- **🟢 Received**: Sample registered
- **🟡 Testing**: Under analysis
- **🟢 Completed**: Testing finished
- **🔴 Rejected**: Failed quality check

## 🆘 Troubleshooting

### "Application won't open"
- ✅ Check all files are in BEMAL folder
- ✅ Try opening in different browser
- ✅ Check file permissions

### "Login not working"

## 🔧 Customization

### Change Laboratory Name
Edit `index.html` line with logo:
```html
<a href="#" onclick="showPage('home')" class="logo">
    🌈 BEMAL
    <span class="lab-badge">Amara Raja</span>
</a>
```

### Add New Sample Types
Edit `script.js` in sample registration:
```javascript
<option value="NewType">New Sample Type</option>
```

### Modify Colors
Edit `styles.css` root variables:
```css
:root {
    --primary-color: #1a365d;  /* Change this */
    --secondary-color: #2d3748; /* Change this */
}
```
