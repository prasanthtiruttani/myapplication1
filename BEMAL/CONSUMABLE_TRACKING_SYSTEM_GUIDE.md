## 🚀 HOW TO USE

### 1. **Login**
- Open the application
- Click "Login" button
- Use credentials from above
- You'll see the Dashboard

### 2. **Dashboard Overview**
Shows key metrics:
- Total Consumables count
- Low Stock Items alert
- Active Requests pending
- Total Suppliers

### 3. **Manage Consumables** 📦
- View all consumables in stock
- Add new consumables with:
  - Name
  - Category (Office Supplies, Stationery, Cleaning, Safety, IT)
  - Stock Level
  - Minimum Stock (alert level)
  - Unit Price
  - Unit (Boxes, Reams, Bottles, Pieces)
- Update stock levels quickly
- Delete items if needed
- See Low Stock warnings in red

**Example Consumables Already Added:**
- Printer Paper A4 (50 reams) - ✅ Available
- Ball Point Pen (15 boxes) - ⚠️ Low Stock Alert
- Hand Sanitizer (8 bottles) - ⚠️ Low Stock Alert

### 4. **Categories** 🏷️
Organize consumables:
- Office Supplies (Paper, ink, folders)
- Stationery (Pens, pencils, notebooks)
- Cleaning (Sanitizers, wipes, disinfectants)
- Safety (Gloves, masks, safety equipment)
- IT (Cables, adapters, storage devices)

**How to add:**
- Click "+ Add Category"
- Enter category name
- Save

### 5. **Suppliers** 🚚
Manage your vendors:
- Add supplier companies
- Store contact person details
- Email and phone numbers
- View all suppliers in one place

**Example Suppliers:**
- Paper World Inc (Contact: John Smith)
- Stationery Hub (Contact: Sarah Johnson)

### 6. **Requests** 📋
Create and approve consumable requests:
- Request consumables from inventory
- Track status (Pending, Approved)
- Add notes for special requirements
- Approver can review and approve

**Example:**
- Printer Paper A4 - Qty: 10 - Status: Pending

## 💡 BEST PRACTICES FOR BEMAL LAB

### 1. **Stock Level Management**
- Set minimum stock levels based on usage
- For frequently used items: Set higher minimum
- For rarely used items: Set lower minimum

### 2. **Regular Updates**
- Update stock after each consumption
- Create requests before running out
- Review low stock items weekly

### 3. **Supplier Management**
- Keep supplier contact details updated
- Add new suppliers as needed
- Track which supplier provides which item

### 4. **Request Workflow**
- Staff creates request
- Approver reviews and approves
- Manager orders from supplier
- Staff receives and updates stock

---

## 🔄 TYPICAL WORKFLOW

```
1. Consumable arrives from store
   ↓
2. Add/Update in CTS system
   ↓
3. Set minimum stock level
   ↓
4. Use consumable in lab
   ↓
5. Stock falls below minimum
   ↓
6. Create request in system
   ↓
7. Approver approves request
   ↓
8. Order from supplier
   ↓
9. Receive new stock
   ↓
10. Update stock level in CTS
    ↓
    Repeat
```
