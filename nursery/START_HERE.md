## 🎓 What You Can Do

### Add a Student
1. Click "Students" in menu
2. Click "+ Add Student"
3. Fill in student details
4. Click "Save Student"
✓ Student appears in table!

### Create a Class
1. Click "Classes" in menu
2. Click "+ Add Class"
3. Fill in class details
4. Click "Save Class"
✓ Class is created!

### Track Fees
1. Click "Fees" in menu
2. Click "+ Add Fee Record"
3. Enter fee details
4. Click "Save Fee"
✓ Fee is tracked!

### Mark Fee as Paid
1. Go to "Fees"
2. Find unpaid fee
3. Click "Mark Paid"
✓ Status changes to "Paid"!
