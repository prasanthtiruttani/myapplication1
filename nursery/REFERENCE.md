2. Click "+ Add Class"
3. Fill form → Save

1. Click "Staff"
2. Click "+ Add Staff"
3. Fill form → Save

### Track Fee
1. Click "Fees"
2. Click "+ Add Fee Record"
3. Fill form → Save

1. Click "Fees"
2. Find unpaid fee
3. Click "Mark Paid"

## ❓ Quick Tips

1. **Lost a record?**
   - Check the table - scroll right if needed
   - Use search/filter if available

2. **Password forgot?**
   - Use demo account
   - Or register a new one

3. **Data looks wrong?**
   - Refresh page (Ctrl+R)
   - Clear browser cache

4. **Want to reset data?**
   - Open browser developer tools (F12)
   - Go to Storage → LocalStorage
   - Delete the entries

5. **Want to customize?**
   - Edit HTML in index.html
   - Edit CSS in styles.css
   - Edit JavaScript in script.js
