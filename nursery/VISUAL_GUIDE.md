## 🔄 User Journey Map

```
FIRST TIME USER
└─ Home Page
   └─ Click "Register"
      └─ Registration Form
         └─ Enter Details
            └─ Click "Register"
               └─ Confirmation Message
                  └─ Redirect to Login Page
                     └─ Login Page
                        └─ Enter Credentials
                           └─ Dashboard
                              ✓ Now can manage nursery

RETURNING USER
└─ Home Page
   └─ Click "Login"
      └─ Login Page
         └─ Enter Credentials
            └─ Dashboard
               └─ View Statistics
                  └─ Manage Data
                     └─ Add/Edit/Delete Records
                        └─ Data Auto-Saved
                           ✓ All operations complete
```

## ⌨️ User Actions Flow

```
LOGIN SUCCESS
├─ User sees: Dashboard with Statistics
├─ Can navigate to: Students, Classes, Staff, Fees
├─ Can perform: Add, Edit, Delete operations
├─ Can logout: Returns to Home
└─ Data auto-saves to localStorage

ADD STUDENT
├─ Click "+ Add Student"
├─ Modal opens with form
├─ Fill all fields
├─ Click "Save Student"
├─ Toast notification appears
├─ Table updates immediately
└─ Data saved to localStorage

MARK FEE PAID
├─ Navigate to Fees
├─ Find unpaid fee
├─ Click "Mark Paid"
├─ Status changes to "Paid"
├─ Status color changes (orange → green)
└─ Dashboard updates
```
